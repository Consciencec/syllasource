Enable .NET Framework from Powershell

Consciencec | Tue 25 August 2020

Download the iso from microsoft, once downloaded right click to mount the disk. Next find the folder soruces/sxs in the mounted iso. Then run below command.

dism /online /enable-feature /featurename:NetFx3 /All /Source:/path/to/sources/sxs /LimitAccess
